
# Submission Form

## About the Project

A simple submission form with basic candidate details. Store the candidate details in the database along with assignment pdf


## Tech Stack Used

**Client:** HTML, CSS, JavaScript, Bootstrap

**Server:** NodeJs, Express

**DataBase:** MongoDB
